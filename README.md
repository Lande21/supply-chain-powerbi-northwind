# supply-chain-powerbi-northwind

Built an interactive dashboard using the Northwind database to analyze sales and supply chain performance.

🚀 Project Overview
- Analyzed 1,000+ records across 8+ relational tables
- Developed 10+ DAX measures for KPI tracking
- Focused on revenue trends, order behavior, and customer insights

🛠 Tools Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Relational Data Modeling
- MongoDB
- MS SQL Server
- Jira
- Canva
- LucidChart
📈 Key Features
- KPI dashboards (Revenue, Orders, Customer Metrics)
- Interactive filters and drill-down capabilities
- Star schema data model for optimized performance and after data integration
📊 Data Modeling
- Integrated Orders, Customers, Products, and Suppliers
- Built relationships to ensure accurate aggregations
- ETL pipeline design
